# coffee_UI_flutter
Simple UI for a coffee app copied off Dribble

## Setup
 a. clone repo `git clone <repo-name>`.

 b. get all dependencies `flutter pub get`.

 c. run the project `flutter run`, `flutter run -d android` or `flutter run -d edge`.
